# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) version 2.1.

## TL;DR

- Be respectful. Assume good faith.
- No personal attacks. No discrimination of any kind.
- Disagree about ideas, not people.
- If something feels off, raise it privately to the maintainers before escalating publicly.

## Reporting

If you experience or witness behaviour that violates this code, contact the maintainer at **brandonrbridges@outlook.com**. Reports are handled confidentially.

## Scope

This applies to all project spaces — GitHub issues and PRs, discussions, Discord (when established), and any other forum where the project is represented.
